Inanimate Object Single Page Design & Build

The Task

The task for this project was to design and develop a solution to promoting a singular product using original content.

The Requirements

The requirments of this project were straight forward, below is a list of them.

  • Original Content
  • Layout Supported Across Devices
  • Semantic Code
  • Work Completed to current standard
  • SASS should be used
  • Well Commented

The Process


The process began by looking at current candle e-commerce websites and seeing what the key details they had easily on show for users to see, using this information, three sketches were produced and had 5-second tests ran on them with users in order to gain an understanding of the design direction that would be taken. This would help define how users would like the design to be carried out.

Wireframing

Wireframes were produced to run the five second tests, this was done to gain some user feedback on some layout concepts, before high-fidelity mock-ups were produced, as this was a small scale project, multiple wireframes could be produced quickly for testing. Below shows one of the wireframes produced. Due to user testing, these wireframes were more detailed than I would normally produce.

The Photography

Taking the photos to produce unique content for the website was initially planned to be conducted like a solo product shoot on clear white backing to remove anything that wasn’t the product, however after perusing this path and discovering it just wasn’t working, the direction was changed to include other items/backing that would help promote features of the candle, the photo shown shows a nice healthy green plant, this promotes the use of natural wax in the candles..

High Fidelity Mock-ups

Having already produced wireframes for initial research, the design phase could move straight into high-fidelity mockup making, the current website that sells this candle doesn’t have much in terms of feel or brand so near complete freedom was allowed with this. The photo shows the final design that was chosen.

The Solution

Ultimately a solution that met all of the requirements and expectations was provided and this project provided a great base for further developing my user research skills and user journey mapping ability.
